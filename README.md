# 💈 Fayola — Frontend

Interface web do sistema de agendamento online para salões, barbearias e estéticas.

## 🔗 Links

- **Landing Page:** https://fayola-frontend-hlfj.vercel.app
- **Painel de Gestão:** https://fayola-frontend-hlfj.vercel.app/login
- **Agendamento Público:** https://fayola-frontend-hlfj.vercel.app/agendar?id=SEU_ID
- **Backend:** https://github.com/liliansantana690-byte/fayola-api

## 🚀 Funcionalidades

- Landing page profissional com planos e funcionalidades
- Tela de login para estabelecimentos
- Dashboard com agenda do dia e receita
- Gestão de serviços e profissionais (cadastrar e excluir)
- Link exclusivo de agendamento para compartilhar com clientes
- Página pública de agendamento com nome do estabelecimento
- Design premium dark gold

## 🛠️ Tecnologias

- React
- Axios
- CSS-in-JS

## ⚙️ Como rodar localmente

1. Clone o repositório
2. Instale as dependências:
```bash
   npm install
```
3. Crie o arquivo `.env`: PORT=3003
REACT_APP_API_URL=http://localhost:3002/api

4. Rode o projeto:
```bash
   npm start
```
